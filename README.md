## üõ†Ô∏è Setting Up Your Kaggle Account

To use kaggle-skill, you need to connect it to your Kaggle account. Here is how:

1. Go to [kaggle.com](https://www.kaggle.com) and log in or create a free account.  
2. Open the "My Account" section from your profile icon.  
3. Scroll to "API" and click "Create New API Token." This downloads a file called `kaggle.json`.  
4. In kaggle-skill, find the "Account Setup" area. You will be asked to upload this file.  
5. Upload `kaggle.json`. This links your software to your Kaggle account securely.  

Once linked, you can access Kaggle datasets, competitions, and notebooks from within the application.
